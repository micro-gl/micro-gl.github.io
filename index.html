micro{gl}
Fast, Super Slim, Embeddable, Headers files only C++11 graphics library, that can run on any 32/64 bits computer without FPU or GPU. No standard library required.
micro{tess}
Fast, Slim, Embeddable, Allocator-aware, Headers files only C++11 geometry tesselation library for vector graphics, that can run on any 32/64 bits computer with or without FPU. No standard library required.
