Introduction

Bezier Patch Tessellator can triangulate

BiQuadrartic
and
BiCubic
2D/3D surfaces.
Triangulates 3d surfaces, but usually used for 2d triangulation with ignoring the z component.

Notes

  1. Output attributes are stored in interleaved format in the supplied number container, example output = [x,y,z,u,v, x,y,z,u,v, x,y,z,u,v, ....]
  2. Output indices of triangulation is stored in a sperate indices container
  3. the type of triangulation is always TRIANGLES_STRIP to save memory
  4. Configurable horizontal/vertical Triangulation resolution
  5. Also interpolates correct UV coords

Algorithm input

Input is

  • meshPoints - array of 9/16 vec3 for quadratic/cubic respectively
  • uOrder/vOrder - should be 3/4 for quadratic/cubic respectively
  • uSamples/vSamples - horizontal/vertical subdivisions count
  • out_vertices_attributes - output container of attributes [x,y,z,u,v,...]
  • out_indices - output container for indices
  • out_indices_type - output triangles type
  • UV bounding box

Example

taken from

examples/example_draw_bezier_patch_raw.cpp

The example demonstrates triangulating a BiCubic surface. I use the `microgl` project to render the tessellation. The algorithm output: - Will fill The `out_vertices_attributes` container, which holds the **(x,y,z,u,v, ...)** - Will fill The `out_indices` container, which holds the triangulation indices - Will fill The `out_indices_type` container, which holds the triangluation type (`TRIANGLES_STRIP` always)

40    using number = float;
41    // using number = Q<16>;
42    using index = unsigned int;
43
44    // microgl drawing setup START
45    using Canvas24 = canvas<bitmap<coder::RGB888_PACKED_32>>;
46    using Texture24 = sampling::texture<bitmap<coder::RGB888_ARRAY>, sampling::texture_filter::Bilinear>;
47    Canvas24 canvas(640, 480);
48    auto img_2 = Resources::loadImageFromCompressedPath("images/uv_512.png");
49    Texture24 tex_uv{new bitmap<coder::RGB888_ARRAY>(img_2.data, img_2.width, img_2.height)};
50    // microgl drawing setup END
51
52    constexpr int samples = 20;
53    constexpr bool debug = true;
54
55    auto draw_bezier_patch = [&](vertex3<number>* mesh) {
56        // Algorithm START
57        // output vertices attributes
58        dynamic_array<number> output_attrib;
59        // output indices
60        dynamic_array<unsigned int> output_indices;
61        // output triangles type
62        microtess::triangles::indices output_indices_type;
63        // define algorithm
64        using tess= microtess::bezier_patch_tesselator<
65                number, number,
66                decltype(output_attrib),
67                decltype(output_indices)>;
68        // compute algorithm
69        tess::compute(mesh, 4, 4, samples, samples, output_attrib,
70                      output_indices, output_indices_type, 0, 0, 1, 1);
71        // Algorithm END
72        
73        canvas.clear({255, 255, 255, 255});
74
75        // walk on pieces of triangles with inverting because it is a TRIANGLES_STRIP
76        using vertex=microtess::vec2<number>;
77        bool even = true;
78        // we alternate order inorder to preserve CCW or CW,
79        for (index ix = 0; ix < output_indices.size()-2; ++ix, even=!even) { 
80            index first_index   = (even ? output_indices[ix + 0] : output_indices[ix + 2]) * tess::ATTRIBUTES_COUNT;
81            index second_index  = (even ? output_indices[ix + 1] : output_indices[ix + 1]) * tess::ATTRIBUTES_COUNT;
82            index third_index   = (even ? output_indices[ix + 2] : output_indices[ix + 0]) * tess::ATTRIBUTES_COUNT;
83
84            vertex p1=vertex{output_attrib[first_index + tess::I_X], output_attrib[first_index + tess::I_Y]};
85            vertex p2=vertex{output_attrib[second_index + tess::I_X], output_attrib[second_index + tess::I_Y]};
86            vertex p3=vertex{output_attrib[third_index + tess::I_X], output_attrib[third_index + tess::I_Y]};
87
88            canvas.drawTriangle<>(
89                    tex_uv,
90                    p1.x, p1.y, output_attrib[first_index  + tess::I_U], output_attrib[first_index  + tess::I_V],
91                    p2.x, p2.y, output_attrib[second_index + tess::I_U], output_attrib[second_index + tess::I_V],
92                    p3.x, p3.y, output_attrib[third_index  + tess::I_U], output_attrib[third_index  + tess::I_V]);
93            if(debug)
94                canvas.drawTriangleWireframe<number>(color_t{0,0,0,255},
95                    {output_attrib[first_index + tess::I_X], output_attrib[first_index + tess::I_Y]},
96                    {output_attrib[second_index + tess::I_X], output_attrib[second_index + tess::I_Y]},
97                    {output_attrib[third_index + tess::I_X], output_attrib[third_index + tess::I_Y]});
98        }
99    };
100
101    auto render = [&](void*, void*, void*) -> void {
102        draw_bezier_patch(bi_cubic_1<number>());
103    };
104
105    example_run(&canvas, render);
106}
