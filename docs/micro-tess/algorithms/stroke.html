Introduction

Stroke Tesselation triangulates a sequence of points into a stroke.

  • Supports dashes
  • Supports stroke caps
  • Supports stroke joins
  • Self contained

Stroke Caps

These can be chosen by the Enum

microtess::stroke_cap

  • stroke_cap::butt
     - No cap is used
  • stroke_cap::square
     - Add a square cap
  • stroke_cap::round
     - Adds a rounded cap

Line Joins

These can be chosen by the Enum

microtess::stroke_line_join

  • stroke_line_join::miter
     - Use miter line join
  • stroke_line_join::miter_clip
     - Use miter line join with clipping (specified later)
  • stroke_line_join::bevel
     - Use bevel line join to fill the gap
  • stroke_line_join::round
     - Use rounded line join to fill the gap

Example

taken from

examples/example_geometry_stroke_dash_tessellation.cpp

The example demonstrates usage of different linear containers (including the packaged non standard

dynamic_array
).
The example also demonstrates different number types such as fixed point integer(
Q
 numbers), float and doubles.
Tessellation computation is only 16 lines of code.

I use the

microgl
project to render the tessellation.
The algorithm output:

  • Will fill The
    output_triangles_type
     for you to know how to iterate the triangles (TRIANGLES/FAN/STRIP)
  • Will fill the user supplied
    output_vertices
     container with triangulation vertices.
  • Will fill the user supplied
    output_indices
     container with triangulation indices.
  • Will (optionally) fill the user supplied
    output_boundary_info
     container for you to know how to identify the boundary.
